Найти рекурсивно текст в файлах каталога (включая вложенные подкаталоги):

```bash
grep -iRl 'ищу текст' .
```

Вывести рекурсивно список файлов в каталоге (включая вложенные подкаталоги):

```bash
find $PWD -type f
```
